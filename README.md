# BlackJack
## 遊戲規則
莊家發給每個人二張牌(包含他自己)，玩家看完後自行決定要不要加，
贏家是在不超過21點的情況下，高於莊家的點數合。

每個玩家的贏法：
1. 拿到五張牌，還不超過21點
2. 掀開牌後合併點數 > 莊家合併的點數

莊家的贏法：
1. 拿到五張牌，還不超過21點
2. 掀開牌後合併點數 > 每個玩家獨立合併的點數

- 2~10 以牌面點數計算
- J、Q、K每張10點
- A可為1或是11點
